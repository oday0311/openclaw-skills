# 抖音视频文字提取工作流使用示例

## 基本用法

### 1. 直接运行工作流

```bash
# 基本用法
python3 scripts/douyin_asr_workflow.py "https://v.douyin.com/xxxxxxx/"

# 显示详细日志
python3 scripts/douyin_asr_workflow.py "https://v.douyin.com/xxxxxxx/" --verbose

# 保存报告到文件
python3 scripts/douyin_asr_workflow.py "https://v.douyin.com/xxxxxxx/" --output report.md
```

### 2. 在OpenClaw中使用

当用户提供抖音分享链接时，自动触发此技能：

```
用户: 帮我提取这个抖音视频的文字内容：https://v.douyin.com/xxxxxxx/

卡卡西AI助手: 好的，我来帮你处理这个抖音视频。让我启动抖音视频文字提取工作流...
```

## 高级用法

### 批量处理

可以编写脚本批量处理多个抖音链接：

```python
#!/usr/bin/env python3
import subprocess

douyin_urls = [
    "https://v.douyin.com/url1/",
    "https://v.douyin.com/url2/",
    "https://v.douyin.com/url3/"
]

for url in douyin_urls:
    print(f"处理: {url}")
    result = subprocess.run([
        "python3", "scripts/douyin_asr_workflow.py",
        url,
        "--output", f"report_{url.split('/')[-2]}.md"
    ])
    
    if result.returncode == 0:
        print(f"成功: {url}")
    else:
        print(f"失败: {url}")
```

### 集成到其他系统

可以将工作流集成到其他系统中，通过API调用：

```python
from douyin_asr_workflow import DouyinASRWorkflow

def process_douyin_video(url):
    workflow = DouyinASRWorkflow(verbose=False)
    results = workflow.run(url)
    report = workflow.generate_report()
    return results, report
```

## 注意事项

1. **网络要求**: 需要稳定的网络连接来下载视频和调用ASR服务
2. **存储空间**: 临时文件会占用磁盘空间，工作流完成后会自动清理
3. **性能考虑**: Whisper第一次运行需要下载模型，后续运行会使用缓存
